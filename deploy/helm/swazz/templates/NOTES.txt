Thank you for installing the Swazz Helm chart!

To access the Web UI:
{{- if .Values.ingress.enabled }}
1. Get the application URL by running these commands:
   echo "Visit https://{{ (index .Values.ingress.hosts 0).host }} to open Swazz"
{{- else }}
1. Port forward the Web UI service:
   kubectl port-forward svc/{{ include "swazz.fullname" . }}-web 8080:{{ .Values.web.service.port }} -n {{ .Release.Namespace }}
   echo "Visit http://localhost:8080 to open Swazz"
{{- end }}

To ensure Runner Agents can connect to the Edge Coordinator, verify that `COORDINATOR_URL` in `runner.env` is accessible by the Runner pods.
